
## 介绍

一个用于开发管理远程服务器的通用框架，实现ssh远程登录的两种方式`密码`和`私钥`，提供了一组可以远程执行命令的`tauri command`，你可以快速接入，开发自己的远程服务器管理工具

## 技术栈:

- Rust、Tauri
- NextJS
